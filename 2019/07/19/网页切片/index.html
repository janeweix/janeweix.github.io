切片可以提高网站性能。网页的打开速度与服务器的响应速度和网页的大小有关，如果图片太大，加载时间太长。所以在此情况下，如果用网页切片，会提高网页的加载速度，增强了用户体验。

切片的技巧

切片越小越好。
渐变、阴影、不规则图形。

阴影
用切片工具选中阴影区域，隐藏阴影区域的背景，导出为 png 格式。

渐变
若为横向渐变，则取高度相同、宽度很小的切片。

不规则图形
框选出图形，隐藏背景，导出为 png 格式。

切片完成后：文件 => 导出 => 存储为web所用格式 => 修改为png格式、设置为透明度。
存储时，设置为“所有用户切片”；指定目录下就会出现 images 文件夹，里面存储了所有的切片。
